# 🚀 High-Performance Telegram Bot on Go (Golang)

[![Go Version](https://shields.io)](https://golang.org)
[![Framework](https://shields.io)](https://github.com)

Архитектурно правильный, высоконагруженный Telegram-бот, написанный на компилируемом языке **Go**. Проект разработан с использованием принципов SOLID и разделения ответственности (Clean Architecture), что позволяет ему выдерживать колоссальные нагрузки при минимальном потреблении ресурсов.

## 🏗 Архитектура проекта (Clean Layout)

Проект разбит на изолированные слои, что исключает циклы импортов и делает код легко расширяемым:

```text
├── cmd/
│   └── bot/
│       └── main.go       # Инициализация и запуск приложения
├── internal/
│   ├── config/
│   │   └── config.go     # Чтение конфигурации и переменных окружения
│   └── telegram/
│       ├── bot.go        # Сердце бота, конфигурация роутера и опций
│       ├── keyboards.go  # Изолированная верстка инлайн-клавиатур (SRP)
│       ├── commands.go   # Хэндлеры для текстовых команд (/start)
│       └── messages.go   # Дефолтный обработчик текста (Fallback/404)
└── config.env            # Файл конфигурации среды окружения
```

## ⚡️ Ключевые преимущества для бизнеса

- **Ультра-низкое потребление RAM:** Бот потребляет всего ~15 МБ оперативной памяти, что позволяет запускать его на самых дешевых серверах (VPS за 150 рублей).
- **Сетка безопасности (Fallback):** Реализован дефолтный роутер. Бот никогда не проигнорирует пользователя, даже если тот ввел неизвестную команду.
- **Гибкая конфигурация:** Адрес Telegram API вынесен в `.env`. Бот готов к бесшовному переключению на локальные сервера или шлюзы в один клик.

## 🛠 Инструкция по развертыванию

1. Склонируйте репозиторий.
2. Создайте файл `config.env` в корне проекта:
   ```env
   BOT_TOKEN=your_telegram_bot_token
   TELEGRAM_API_URL=https://telegram.org
   ```
3. Запустите проект из корневой директории:
   ```bash
   go run cmd/bot/main.go
   ```
